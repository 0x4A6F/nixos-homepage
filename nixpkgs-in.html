Nix Packages collection

The Nix Packages collection (nixpkgs) is a distribution of Nix expressions (= Nix package descriptions) for hundreds of existing Unix components, ranging from fundamental components like GCC and the C library Glibc to user applications like Firefox and Eclipse.

Latest stable release

The most recent stable release is Nixpkgs 0.11. There is a list of changes in this release.

Unstable releases

Nixpkgs is also built automatically from its Subversion repository. You can download the latest pre-release, or view a list of all recent pre-releases built in our build farm.

Contributing

The Nix manual has information about writing Nix expressions.

To have your packages added to Nixpkgs, either send your patches to the mailing list, or ask for commit access to the Subversion repository.
